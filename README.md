# POWER-BI--ANALISIS-DE-PATENTES
# 📊 Análisis de Patentes – Power BI (30 años)

## 📌 Descripción del proyecto

Este proyecto presenta un análisis visual e interactivo sobre el comportamiento de las **patentes registradas durante un periodo de 30 años**, desarrollado utilizando **Power BI**.

El objetivo principal es explorar tendencias históricas, clasificación tecnológica y distribución geográfica de las patentes mediante dashboards dinámicos que faciliten la toma de decisiones y el análisis económico-tecnológico.

---

## 🎯 Objetivos

- Analizar la evolución temporal del registro de patentes.
- Identificar las principales clasificaciones tecnológicas (CRC).
- Visualizar la participación por categorías.
- Analizar la distribución geográfica de las patentes.
- Facilitar la exploración de datos mediante filtros interactivos.

---

## 🧰 Tecnologías utilizadas

- Power BI Desktop  
- Modelado de datos  
- DAX (Data Analysis Expressions)  
- Visualización interactiva de datos  
- Mapas geoespaciales (Bing Maps)

---

## 📈 Principales visualizaciones

El dashboard incluye:

- Indicador de **total de patentes** analizadas.
- Conteo de patentes por año (serie temporal).
- Participación por clasificación tecnológica (gráfico de pastel).
- Mapa interactivo con ubicación geográfica.
- Filtros dinámicos:
  - Año
  - Clasificación de patentes
  - Ciudad
